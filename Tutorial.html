Modded Switch

To start with, you need a modded Switch. If your Switch is not modded, you can follow this guide.
Make sure you follow all the steps and don't miss anything, otherwise you might fail in the process and render your Switch useless.

Downloading Super Luigi Odyssey

Once you have a modded Switch, download Super Luigi Odyssey in the main page.
Remember that you need a copy of Super Mario Odyssey in your Switch, either digital or physical. If you have the latest version of Super Mario Odyssey, download v1.1 of Super Luigi Odyssey, else, download v1.0.

Installing Super Luigi Odyssey

Extract the contents of the Super Luigi Odyssey ZIP file into the following path on your Switch's SD card:

/atmosphere/contents/0100000000010000/romfs

If you are using an older version of Atmosphere to load the custom files, use the following path instead:

/atmosphere/titles/0100000000010000/romfs

Booting Atmosphere and running Super Luigi Odyssey

To launch Super Luigi Odyssey, you need to boot into RCM mode (if you have a modded Switch you should know how to do this). Make sure you launch the CFW payload with Atmosphere. To know if the payload correctly launched Atmosphere, launch Super Mario Odyssey. If it's changed to Super Luigi Odyssey (note that the icons on the home menu still say Super Mario Odyssey), congratulations! If not, please try again with a different payload.
